Framework, Twitter Bootstrap!

Bootstrap v.2.n es una herramienta para el desarrollo rápido y correcto de aplicaciones y sitios web. Fomenta las buenas prácticas de diseño y desarrollo web, conforme a estándares W3C. Permite diseñar webs adaptables y fluidas, visualizables correctamente en múltiples dispositivos (Responsive Web Design). Incluye una robusta base de HTML5, CSS3 y Javascript, también incluye elementos de diseño, tipografías, tablas, formularios, navegación, alertas, etc.

Creado por y para Geeks.

Como tú, nos encanta crear productos espectaculares en la web. Nos gusta tanto, que decidimos ayudar a la gente como nosotros para hacerlo más fácil y más rápido. Boostrap está creado para ti.

Para todo tipo de niveles

Boostrap está diseñado para ayudar a la gente de todos los niveles – diseñador o desarrollador, un gran master o un principiante. Utilízalo como un conjunto completo o para empezar algo más complejo.

Cruzando los limites

En principio creado solo teniendo los navegadores modernos en mente, Boostrap ha evolucionado para incluir soporte para un mayor número de navegadores (incluso IE7), y, con Boostrap 2, también para tabletas y smartphones.

Rejilla(Grids) de 12 columnas

Los sistemas de rejillas (grids) no harán todo el trabajo, pero usando una rejilla como esqueleto harán tu trabajo más cómodo y simple. Usa nuestras clases para grids para crear o empieza la tuya propia.

Diseño Adaptable

Con Bootstrap 2. nuestros componentes se adaptan y escalan según las resoluciones de pantalla y dispositivos para proporcionar una experiencia consistente, no importa el qué.

Documentación de estilos

A diferencia de otros conjuntos de herramientas de front-end, Boostrap primero se diseñó más como una guía de estilo para documentos no sólo con nuestras características, pero mejora la práctica y la vida.

Biblioteca en aumento

Aunque sólo pesa 10kb (gzipped), Boostrap es uno de los conjuntos de herramientas más completos con docenas de componentes funcionales completos listo para poner en uso.

Personalizar Plugins jQuery

¿Qué tiene de bueno un componente de diseño espectacular sin interacciones listas para usar, adecuadas y extensibles? Con Boostrap, puedes crear tu plugin de jQuery personalizado para dar vida a tus proyectos.

Crear en LESS

Donde CSS falla, LESS destaca. Variables, anidaciones, operaciones y mezclas en LESS hace la codificación de CSS más rápida y más eficiente con un mínimo esfuerzo.
