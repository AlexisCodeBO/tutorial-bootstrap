¿Qué es el Framework Bootstrap de Twitter?

Bootstrap es una herramienta para el desarrollo rápido y correcto de aplicaciones y sitios web. Fomenta las buenas prácticas de diseño y desarrollo web, conforme a estándares W3C. Permite diseñar webs adaptables y fluidas, visualizables correctamente en múltiples dispositivos (Responsive Web Design). Incluye una robusta base de HTML5, CSS3 y Javascript, también incluye elementos de diseño, tipografías, tablas, formularios, navegación, alertas, etc.

En esta página y en consecuencia, el tutorial cubre herramienta de Twitter Bootstrap, en detalle, lo que le preparará para construir aplicaciones web y sitios que utilizan la herramienta.

Soporte del navegador
Bootstrap se ha creado con los navegadores modernos en mente. Por lo tanto, se encuentra funcionando a la perfección con todas las versiones modernas de Chrome, Firefox, Safari, Opera e Internet Explorer.

Obtener Twitter Bootstrap
Descargar Bootstrap de Twitter desde https://github.com/twitter/bootstrap.

Descomprimir el archivo descargado y obtendrá todos los archivos necesarios para trabajar con el kit de herramientas.

Primeros pasos con Bootstrap
Puede incluir Twitter Bootstrap en su página HTML de dos maneras.

Usted simplemente puede incluir

<link rel="stylesheet" href="/twitter-bootstrap/twitter-bootstrap-v2/docs/assets/css/bootstrap.css">
en la sección de la cabeza de su archivo HTML.

Desde Bootstrap soporta menos (preprocesador CSS para el desarrollo web más rápido y más fácil), puede incluir el archivo CSS menos y menos archivo JS en su página HTML, como

rel="stylesheet/less" <link href="/path/to/bootstrap.less">
<script src="/path/to/less.js"> </ script>
También puede utilizar Bootstrap NetDNA de CDN @ http://www.bootstrapcdn.com/

Lo que usted aprenderá
En el tutorial "Bootstrap Tutorial de Twitter", usted aprenderá lo siguiente.

Sistema de red

Usted verá cómo crear sistema de cuadrícula utilizando Twitter Bootstrap. Es compatible con 940px de ancho 12 de cuadrícula de columna por defecto y muchas variaciones de la misma.

Presentaciones

Usted verá cómo crear 940px de ancho fijo y Diseño de cómo crear diseños fluidos. Esto le ayudará a crear plantillas HTML con facilidad.

Tipografía

Usted verá cómo crear encabezados, párrafos, listas y otros elementos de tipo inline.

Tablas

Usted verá cómo construir tablas para contener los datos tabulares.

Formularios

Usted verá cómo construir formas elegantes, incluyendo texto, botón de radio, área de texto, diversos tipos de botones en la página HTML.

Navegación

Usted verá cómo construir la navegación moderna y elegante. Y también la paginación.

Alertas y error

Como diseñador de páginas web / desarrollador a menudo es necesario crear alertas de los errores, las advertencias, etc En este tutorial vamos a aprender a construir los que utilizan Twitter Bootstrap.

Popovers

Usted aprenderá a utilizar la herramienta para construir las ventanas modales, información sobre herramientas y panecillos.
