Diseñado para todo el mundo, en todas partes.

¿Necesitas razones para querer Boostrap? No busques más.

Creado por y para nerds

Como tú, nos encanta crear productos espectaculares en la web. Nos gusta tanto, que decidimos ayudar a la gente como nosotros para hacerlo más fácil y más rápido. Boostrap está creado para ti.

Para todo tipo de niveles

Boostrap está diseñado para ayudar a la gente de todos los niveles – diseñador o desarrollador, un gran nerd o un principiante. Utilízalo como un conjunto completo o para empezar algo más complejo.

Cruzando los limites

En principio creado solo teniendo los navegadores modernos en mente, Boostrap ha evolucionado para incluir soporte para un mayor número de navegadores (incluso IE7), y, con Boostrap 2, también para tabletas y smartphones.

Grids de 12 columnas

Los sistemas de grillas (grids) no harán todo el trabajo, pero usando una grilla como esqueleto de tu trabajo harán tu trabajo más simple. Usa nuestras clases para grids para crear o empieza la tuya propia.

Diseño Adaptable

With Bootstrap 2. nuestros componentes se adaptan y escalan según las resoluciones de pantalla y dispositivos para proporcionar una experiencia consistente, no importa el qué.

Documentación de estilos

A diferencia de otros conjuntos de herramientas de front-end, Boostrap primero se diseñó más como una guía de estilo para documentos no sólo con nuestras características, pero mejora la práctica y la vida.

Biblioteca en aumento

Aunque sólo pesa 10kb (gzipped), Boostrap es uno de los conjuntos de herramientas más completos con docenas de componentes funcionales completos listo para poner en uso.

Personalizar Plugins jQuery

¿Qué tiene de bueno un componente de diseño espectacular sin interacciones listas para usar, adecuadas y extensibles? Con Boostrap, puedes crear tu plugin de jQuery personalizado para traer tus proyectos a la vida.

Crear en LESS

Donde CSS falla, LESS destaca. Variables, anidaciones, operaciones y mezclas en LESS hace la codificación de CSS más rápida y más eficiente con un mínimo esfuerzo.

HTML5
